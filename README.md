# 零食热量计算器

一个功能丰富的零食热量查询与健康管理工具，帮助用户了解零食热量、计算运动消耗、获取健康替代品推荐，并支持个人热量记录追踪。

## 项目简介

零食热量计算器是一款面向健康饮食关注者的前端应用。用户可以查询各种零食的热量信息，了解需要多少运动才能消耗掉这些热量，获取更低热量的替代品推荐，并记录每日热量摄入与消耗，支持 BMI 计算和个性化健康建议。

### 主要特性

- 🔍 **零食搜索**：支持按名称、分类、标签多维度搜索
- 🔥 **热量查询**：详细的营养成分展示（蛋白质、脂肪、碳水）
- 🏃 **运动消耗**：基于 MET 代谢当量计算多种运动的消耗时间
- 💡 **智能推荐**：基于口味相似度和热量差的替代品推荐算法
- 📝 **热量记录**：每日饮食摄入与运动消耗记录追踪
- ❤️ **收藏管理**：支持自定义分类收藏和自定义标签
- 📊 **数据可视化**：营养成分饼图、日历热力图、每日报表
- 🧮 **BMI 计算**：身体质量指数计算与健康建议
- 🏋️ **运动方案**：多种难度的个性化运动方案生成

## 核心算法说明

### 1. MET 运动消耗计算公式

```
每分钟消耗热量（kcal/min）= (MET值 × 3.5 × 体重(kg)) / 200

运动时间（分钟）= 零食热量(kcal) / 每分钟消耗热量
```

- MET（Metabolic Equivalent of Task）：代谢当量，表示相对于静坐时的能量消耗倍率
- 默认体重：65kg
- 误差范围：约 ±15%

### 2. 替代品推荐评分算法

```
综合评分 = 口味相似度 × 0.35 + 营养相似度 × 0.25 + 健康评分 × 0.2 + 分类加分
```

| 因素       | 权重 | 说明                            |
| ---------- | ---- | ------------------------------- |
| 口味相似度 | 35%  | 基于标签的 Jaccard 相似度       |
| 营养相似度 | 25%  | 热量/蛋白/脂肪/碳水的归一化差异 |
| 健康评分   | 20%  | 热量减少比例，最高 0.5 分       |
| 同类加分   | 20%  | 同类零食额外加 0.2 分           |

### 3. BMI 计算公式

```
BMI = 体重(kg) / (身高(m))²
```

| BMI 范围  | 分类     | 零食热量限制 |
| --------- | -------- | ------------ |
| < 18.5    | 偏瘦     | 500 kcal/天  |
| 18.5 - 24 | 正常     | 300 kcal/天  |
| 24 - 28   | 偏胖     | 200 kcal/天  |
| 28 - 32   | 肥胖     | 150 kcal/天  |
| ≥ 32      | 重度肥胖 | 150 kcal/天  |

### 4. 每日热量需求计算（Mifflin-St Jeor 公式）

```
男性: BMR = 88.362 + (13.397 × 体重) + (4.799 × 身高) - (5.677 × 年龄
女性: BMR = 447.593 + (9.247 × 体重) + (3.098 × 身高) - (4.330 × 年龄

每日所需 = BMR × 活动系数
```

| 活动水平 | 活动系数 |
| -------- | -------- |
| 久坐     | 1.2      |
| 轻度活动 | 1.375    |
| 中等活动 | 1.55     |
| 高度活动 | 1.725    |
| 非常活跃 | 1.9      |

## 优化与迭代方向

### 1. 数据层优化

#### 1.1 数据丰富度提升

- **扩展零食数据库**：目前仅有 24 种零食，建议扩充到 100+ 种，覆盖更多品类（如奶茶、咖啡、烘焙、快餐等）
- **添加食物图片**：目前使用 emoji 占位，建议替换为真实食物图片
- **营养数据细化**：增加膳食纤维、糖、钠、胆固醇等更多营养指标
- **分量换算**：支持不同分量（如 100g / 一份 / 一包）的自动换算

#### 1.2 数据持久化升级

- **后端服务接入**：从纯前端静态数据 → 后端 API 服务

- **数据缓存**：结合 IndexedDB 本地缓存 + 服务端同步
